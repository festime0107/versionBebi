# Bebi Musliman v3

Projekt Next.js për bebe me seksione:
- Kafshët
- Numrat 1–20
- Gjumë
- Ngjyrat
- Format
- Familja
- Veprime të mira
- Ushqimet
- Transporti

## Si ta përdorësh
```bash
npm install
npm run dev
```

## Audiot
Brenda `public/audio/...` ke folderat gati.
Placeholder-at janë `.gitkeep` dhe `README.txt`.

Kur të shtosh MP3-të e tua:
- lëri folderat siç janë
- fshi vetëm placeholder-at nëse do
- vendos MP3 me emrat e saktë që përdoren në `app/page.js`
